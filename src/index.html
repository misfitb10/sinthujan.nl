Smartwatch app

Naast mijn schoolprojecten wilde ik een ontwerp maken voor een app. Een ontwerp maken voor een smartwatch vond ik meer uitdagend. Mijn concept gaat over vrienden uitdagen om meer te laten bewegen. Laatste tijd wordt vakar stappenteller gebruikt. Met deze app kan de gebruiker zijn/haar vrienden uitdagen via smartwatch om te kijken wie meer stappen zet. Meeste mensen willen graag winnen. Hierdoor zijn mensen geneigd om te bewegen.

  • Concept bedenken
  • Visualiseren

Parkwijzer

Gezamelijk met medestudenten (projectgroep) moest ik voor schoolproject een mobiele app ontworpen om parkeerproblemen bij automobilisten op te lossen. Het doel van dit project was om te onderzoeken welke mogelijkheden er bestaan om het parkeerprobleem op te lossen met behulp van navigatiesystemen. Door middel van kwalitatief onderzoek werd er een concept ontwikkeld waarmee een gebruiksvriendelijk ontwerp gerealiseerd kan worden.
